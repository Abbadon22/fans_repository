# Fans Launcher — 7 Days to Die

Лаунчер для игроков закрытой группы Fans: проверка модов, установка модпака с сервера, запуск игры и подключение к нашему серверу.

> **Скачивайте только установщик** с официальной страницы релизов. Сборка из исходников игрокам не нужна.

## Скачать

**[Последняя версия (GitHub Releases)](https://github.com/Abbadon22/fans_repository/releases/latest)**

1. Откройте ссылку выше.
2. Скачайте файл **`Fans Launcher_*_x64-setup.exe`**.
3. Установите и запустите **Fans Launcher**.

Уже установленная копия обновляется **сама** при запуске (через GitHub Releases).

## Что нужно на компьютере

- Windows 10/11
- **Steam-версия** [7 Days to Die](https://store.steampowered.com/app/251570/)
- Интернет (моды качаются с Яндекс.Диска по списку с сервера)

Node.js, Rust и прочие инструменты разработчика **не требуются**.

## Первый запуск

1. Запустите **Fans Launcher**.
2. На вкладке **«Главная»** нажмите **«Выбрать папку»** и укажите каталог Steam с файлом `7DaysToDie.exe`  
   (часто: `Steam\steamapps\common\7 Days To Die`).
3. Лаунчер **проверит** моды. Если чего-то не хватает — откройте **«Моды»** и нажмите **«Установить N мод(ов)»**.
4. Когда статус **«Готово к запуску»** — нажмите **«Играть»**.  
   По умолчанию Steam откроет подключение к серверу (можно отключить в **«Настройки»**).

## Вкладки

| Вкладка | Зачем |
|--------|--------|
| **Главная** | Статус, сервер, журнал событий |
| **Моды** | Список модпака, проверка, установка недостающих |
| **Настройки** | Пароль сервера, папка игры, обновления лаунчера |

## Моды

- Список модов задаётся на сервере (`manifest.json`). Лаунчер подтягивает его автоматически.
- Скачиваются **только** отсутствующие или устаревшие моды, не весь пакет целиком.
- Папки в `Mods/`, которых **нет** в актуальном списке, **удаляются** при проверке (лишние моды с прошлых сборок).

Папка модов в игре: `<ваша игра>\Mods\`.

## Сервер

| Параметр | Значение |
|----------|----------|
| Адрес | `epyc2.worldhosts.fun:27681` |
| Пароль | задаётся в **Настройки** (сохраняется в `config.json` рядом с лаунчером) |

Скопировать IP или ссылку Steam можно на **Главной**.

## Обновления

| Что | Как обновляется |
|-----|------------------|
| **Лаунчер** | Автоматически при запуске (или «Проверить» в Настройках) |
| **Список модов** | С сервера, без переустановки лаунчера |
| **Файлы модов** | Кнопка установки на вкладке «Моды» |

## Частые вопросы

**Лаунчер не находит игру**  
Укажите папку, где лежит `7DaysToDie.exe`, а не папку `Mods` и не ярлык на рабочем столе.

**Моды не качаются**  
Проверьте интернет и вкладку «Моды» → «Обновить список». Смотрите **Журнал** на главной.

**Игра запущена, но «Играть» неактивна**  
Закройте `7DaysToDie.exe` и снова нажмите «Играть» в лаунчере.

**Нужна помощь**  
Сохраните журнал (**«Сохранить»** на главной) и отправьте администратору группы.
